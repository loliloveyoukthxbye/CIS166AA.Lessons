The modules I reviewed in this week's lesson was a fantastic refresher of what I learned about JavaScript in my first web development course (CIS133DA). We reviewed the basic components of JavaScript which included defining variables, events and objects. We covered how to comment when building out our JavaScript code and how to open the console in the web browser to run JavaScript code and see the results in the browser right away. The most important lesson I've learned this week was JavaScript coding standards which I am still working to commit to memory.

JavaScript is an extremely useful tool to use in web development because it can be used to add interactive features to the webpage. We can use functions, for example, to add features like pop-up form validations or updates like "last updated" with a timestamp at the bottom of a webpage. We can use loops to tell the program to repeat instructions, for example, we can create an interactive feature like an animation with a loop.
